Pokemondo

How to Play

  1. There are two players. The player whose turn it is throws a pokeball. Here is the basic rules of each pokeball they can open:
  2. If a player throw a pickachu they get electrocuted which results in -3 points.
  3. If a player throw a charzard it’s flames will help keep you warm which results in +1 point.
  4. If a player throw a jigglypuff it will make you sleepy which results -1 point.
  5. If a player throw a snorlax it’s hug will make you happy which results in +3 points.
  6. If a player throw a ho-oh the rainbow brings wealth to you which results in +5 points.
  7. After each throw, the current player can either throw again, (at the risk of possibly loosing points) or if the current player feels that luck is running thin, they can pass to the other player.
  8. The first player to get 30 points or higher wins.
